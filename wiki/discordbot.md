# Discord Bot

A Discord bot is a special Discord user that can do certain things. you can operate them with code.
They can be coded in many languages using Wrapper libraries. These are:

- orca	          C
- Discord.Net	    C#
- DSharpPlus	    C#
- Remora.Discord	C#
- aegis.cpp	      C++
- D++           	C++
- Discord++	      C++
- Sleepy Discord	C++
- discljord	      Clojure
- discordcr	      Crystal
- nyxx	          Dart
- coxir	          Elixir
- Nostrum	        Elixir
- DiscordGo	      Go
- DisGord	        Go
- catnip	        Java
- Discord4J	      Java
- Javacord	      Java
- JDA	            Java
- discord.js	    JavaScript
- Eris	          JavaScript
- Discord.jl	    Julia
- Discordia	      Lua
- lacord	        Lua

- DiscordPHP	    PHP
- RestCord	      PHP
- discord.py	    Python
- disco	          Python
